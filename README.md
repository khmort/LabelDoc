# Document Labeling Tool for YOLO

## Description
This project is a dedicated application for **document labeling** with output in **YOLO format**.  
It allows users to draw bounding boxes on different parts of scanned documents and generate annotations ready for object detection training.  
![LabelDoc Demo][demo.gif]

---

## ⚡ Features
- Designed specifically for **document labeling**
- YOLO-compatible output format
- **Smart auto-correction of bounding boxes** to speed up the labeling process
- Simple UI for drawing bounding boxes

---

## ⚠️ Limitations
- Currently works **only on Linux**
- The source code is in an **early stage**, with non-standard naming and messy structure
- Future versions will include refactoring and improvements
